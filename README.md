# Security-Header-Checker
### A simple tool used to analyze security headers and identify vulnerabilities!


#### Features
- Analyzes security headers of websites you visit
- Shows you which security protections are enabled or missing
- Calculates a security score for the current page

#### Additional Notes
- As of 01/18/25, this extension is now available for public use in the google chrome extension store. Hooray!
- The main permission this extension uses is "activeTab". No data is being sold and is overall not invasive.

#### Uses
- I hope for everyone to check out their favorite sites, document their safety and integrity
- Perhaps adapt to this extension in your own way!
